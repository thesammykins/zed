rulename: Use TODO.md
rule:
When working on long-form tasks, ensure you setup a TODO.md in the root directory of the project.

Setup the file with the structure below:

# TODO
SUMMARY OF TASK

- [ ] Task 1
- [ ] Task 2
- [ ] Task 3

When completing tasks, mark them as done by adding an x in the square brackets:

# TODO
SUMMARY OF TASK

- [x] Task 1
- [x] Task 2
- [x] Task 3

When you are done with all tasks and there is a valid SUMMARY.md in the root of the project, you can delete the TODO.md file.
