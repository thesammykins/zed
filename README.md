# Zed Editor Configuration

This repository contains my personal configuration and rules for the Zed editor.

## Contents

### Rules (`rules/`)
- **`.template.md`** - Template for creating new rule files
- **`generatesummary.md`** - Rules for generating project summaries
- **`initproject.md`** - Project initialization guidelines
- **`memorybankmcp.md`** - Memory bank MCP integration rules
- **`prcomments.md`** - Pull request comment automation
- **`prreview.md`** - Pull request review guidelines
- **`todo_create.md`** - TODO creation and management rules

### Settings
- **`settings.json`** - Zed editor configuration settings

## Usage

These files are designed to work with Zed's rule-based automation system to streamline development workflows and maintain consistent coding practices.

## Installation

1. Clone this repository
2. Copy the relevant files to your Zed configuration directory
3. Customize the rules according to your preferences
